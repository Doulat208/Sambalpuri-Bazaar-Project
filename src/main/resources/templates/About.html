About Sambalpuri Bazaar

Sambalpuri Bazaar is a web-based e-commerce admin and product management system designed to promote authentic Sambalpuri handloom products and traditional Odia culture. The project focuses on efficient product management, user handling, and a clean user interface using modern web technologies.
